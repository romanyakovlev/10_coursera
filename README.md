# 10_coursera

Скрипт, который берет 20 случайных курсов с курсеры, парсит их и записывает информацию в xlsx.

# Подготовка

Устанавливаем необходимые библиотеки:
- beautifulsoup4
- requests
- openpyxl 

```sh
pip install -r requirements.txt
```

# Запуск

Запускаем скрипт

```sh
python3 coursera.py
```
